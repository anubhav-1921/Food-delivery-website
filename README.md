# 🍔 Foodie - Responsive Food Delivery Website

A modern and fully responsive **Food Delivery Website** built using **HTML, CSS and JavaScript**. The project features an attractive UI, interactive shopping cart, mobile-friendly navigation, product slider, customer reviews, and a smooth user experience.

## 🚀 Live Demo

👉 https://anubhav-1921.github.io/Food-delivery-website/

---

## ✨ Features

- Responsive Design
- Modern Landing Page
- Interactive Shopping Cart
- Add to Cart Functionality
- Quantity Increase/Decrease
- Automatic Total Price Calculation
- Customer Review Slider
- Mobile Navigation Menu
- Clean UI/UX
- Smooth Animations
- Swiper.js Slider

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Swiper.js
- Font Awesome
